Vulnerable Test Application

This is a simple test target for WAF testing.

Test Endpoints:

  • API Endpoint
  • Search (SQL Injection Test)
  • Comment Form (XSS Test)
  • Ping Utility (Command Injection Test)

Sample Attack Patterns:

  • SQL Injection: /search.php?q=1' OR '1'='1
  • XSS: /comment.php?comment=<script>alert('XSS')</script>
  • Command Injection: /ping.php?host=127.0.0.1; cat /etc/passwd
  • Path Traversal: /file.php?path=../../../../etc/passwd
